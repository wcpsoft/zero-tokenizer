# 许可证说明

本项目采用双重许可证 - 您可以选择使用 Apache 2.0 许可证或 MIT 许可证。

## Apache 2.0 许可证

Apache 2.0 是一种宽松的开源许可证，允许用户自由使用、修改、分发和销售软件，同时保留版权声明。

- 英文版本：[LICENSE-APACHE-2.0-EN](LICENSE-APACHE-2.0-EN)
- 中文版本：[LICENSE-APACHE-2.0-ZH](LICENSE-APACHE-2.0-ZH)

## MIT 许可证

MIT 许可证是一种非常宽松的开源许可证，允许用户几乎无限制地使用、修改和分发软件。

- 英文版本：[LICENSE-MIT-EN](LICENSE-MIT-EN)
- 中文版本：[LICENSE-MIT-ZH](LICENSE-MIT-ZH)

## 选择许可证

您可以根据您的需求选择以下任一许可证：

1. 如果您需要更明确的专利授权条款，建议选择 Apache 2.0 许可证
2. 如果您需要更简洁的许可证条款，建议选择 MIT 许可证

我们强烈建议您在无论任何许可证的条件下，请声明参考了本项目的代码并附上原始代码的链接。
